# data/newpipe — NewPipe Extractor 격리 패키지

이 패키지는 SubFeed 가 NewPipe Extractor 라이브러리에 의존하는 **유일한** 코드 영역이다. NewPipe 버전 업데이트 시 이 폴더와 빌드 설정 (`gradle/libs.versions.toml`, `app/build.gradle.kts`, `app/proguard-rules.pro`) 만 확인하면 된다.

업데이트 절차는 `docs/newpipe/maintenance.md` 와 `/sync-newpipe` 명령 참고.

---

## 파일 구성

| 파일 | 책임 | NewPipe API 의존도 |
|---|---|---|
| `SubFeedDownloader.kt` | NewPipe 의 `Downloader` interface 구현 — OkHttp wrapper | 낮음 (interface 안정적) |
| `NewPipeStreamFetcher.kt` | `YoutubeStreamExtractor` 호출, `StreamInfo` 변환 | **높음 (메인 API surface)** |
| `DashMpdBuilder.kt` | `AudioStream`/`VideoStream`/`ItagItem` → DASH MPD XML | 높음 (stream 모델 의존) |

`NewPipeVideoExtractor.kt` (한 단계 위 패키지) 는 thin orchestrator 로 NewPipe 와 무관.

## 우리가 의존하는 NewPipe API (v0.26.2 기준)

업데이트 시 NewPipe Extractor 의 release notes / migration guide 와 이 목록을 대조한다.

### Bootstrap (SubFeedApp.onCreate)

```kotlin
NewPipe.init(downloader: Downloader, localization: Localization)
```

스트림 추출은 v0.26.3+ 의 visionOS 클라이언트(PoToken 불요)에 의존하므로 `setPoTokenProvider` 는 호출하지 않는다.
WEB-context PoToken 은 NewPipe 가 쓰는 ANDROID/IOS/visionOS 스트림 클라이언트와 영구 불일치라 무용지물 (`docs/20260529_youtube-potoken-quality-wall.md` 참고).

### 서비스 진입점 (NewPipeStreamFetcher)

```kotlin
ServiceList.YouTube                                              // StreamingService
ServiceList.YouTube.streamLHFactory.fromUrl(String): LinkHandler
ServiceList.YouTube.getStreamExtractor(LinkHandler): StreamExtractor
```

반환된 `StreamExtractor` 를 `YoutubeStreamExtractor` 로 캐스팅해서 사용.

### YoutubeStreamExtractor 인스턴스 (NewPipeStreamFetcher)

```kotlin
fetchPage(): Unit                                  // PoToken + InnerTube + base.js 한 번에
length: Int                                        // duration in seconds (or 0)
description: Description?                          // .content: String
dashMpdUrl: String                                 // n-deobfuscated DASH manifest URL (or empty)
hlsUrl: String                                     // HLS manifest URL (live 등, or empty)
videoOnlyStreams: List<VideoStream>                // adaptive video (no audio)
audioStreams: List<AudioStream>                    // adaptive audio
videoStreams: List<VideoStream>                    // muxed (last resort)
subtitlesDefault: List<SubtitlesStream>            // 자막
```

### SubtitlesStream (NewPipeStreamFetcher.collectCaptions)

```kotlin
languageTag: String?
content: String                                    // 자막 URL
isAutoGenerated: Boolean
```

### AudioStream / VideoStream 공통 (DashMpdBuilder)

```kotlin
content: String                                    // n-deobfuscated chunk URL (BaseURL 로 박음)
itagItem: ItagItem?                                // null 이면 skip
codec: String?
format: MediaFormat?                               // .mimeType
```

### VideoStream 전용

```kotlin
width: Int
height: Int
fps: Int                                           // 0 이면 표기 생략
```

### AudioStream 전용

```kotlin
averageBitrate: Int                                // 0 이면 itagItem.avgBitrate, 그것도 0 이면 default 128000
audioLocale: Locale?                               // .language: String?
audioTrackId: String?                              // "ko.4" 등에서 lang 코드 추출
```

### ItagItem (DashMpdBuilder)

```kotlin
bitrate: Int                                       // 0 이면 avgBitrate, 그것도 0 이면 default 1_000_000
avgBitrate: Int
sampleRate: Int                                    // 0 이면 default 44100
getInitStart(): Int                                // DASH SegmentBase init range
getInitEnd(): Int
getIndexStart(): Int                               // DASH SegmentBase index range
getIndexEnd(): Int
```

### Downloader (SubFeedDownloader)

```kotlin
abstract class Downloader {
    @Throws(IOException::class, ReCaptchaException::class)
    abstract fun execute(request: Request): Response
}

class Request {
    fun httpMethod(): String
    fun url(): String
    fun headers(): Map<String, List<String>>
    fun dataToSend(): ByteArray?
}

class Response {
    constructor(code: Int, message: String, headers: Map<String, List<String>>, body: String?, latestUrl: String)
}
```

### Localization / 기타

```kotlin
Localization(languageCode: String, countryCode: String)
ContentCountry(countryCode: String)        // 현재 사용 안 함, import 만 남아있음
ReCaptchaException                          // Downloader 시그니처
```

## 업데이트 시 의사결정 가이드

NewPipe release notes 를 이 목록과 대조해 영향 등급 매긴다:

| 변경 종류 | 영향 등급 | 우리 작업 |
|---|---|---|
| 위 목록에 없는 API 변경 | 영향 없음 | 그냥 버전만 올림 |
| 위 목록 메서드의 시그니처 변경 | 컴파일 깨짐 | 해당 caller 수정 (이 폴더 안) |
| `Downloader` interface 변경 | 컴파일 깨짐 | `SubFeedDownloader` 수정 |
| `ItagItem` deprecation | 위험 | 대체 API 조사 후 `DashMpdBuilder` 수정 |
| Rhino 버전 점프 | 위험 | ProGuard 룰 재점검 |
| protobuf 버전 변경 | 위험 | `app/build.gradle.kts` 의 exclude/force 전략 재점검 (Firebase BOM 매치) |

빌드 깨지면 컴파일 에러 위치는 거의 확실하게 이 폴더 안. 깨진 메서드를 NewPipe 본가 코드 (https://github.com/TeamNewPipe/NewPipeExtractor) 에서 검색해 대체 API 확인.
